Résumé Cours JavaScript :

- Déclarer une variable et lui attribuer un contenu :

➞ var nom_variable = contenu; // déclarer une variable et lui attribuer un contenu ( texte , nombre , nombre flottant , boolean ect...)

➞ let nom_variable = contenu; // comme avec var mais avec une portée de bloc plus stricte

➞ variable contenu_2; //réattribuer un nouveau contenu à une variable déja existante

➞ var nom_variable = prompt("question - text ); //ecrire du contenu dans pop up pour ensuite la attribuer à la variable ( contenu texte uniquemant )

---- nom_variable = parseInt(nom_variable); // pour rendre le contenu du prompt en format nombre et non format texte

---- nom_variable = parseFloat(nom_variable); // pour rendre le contenu du prompt en format nombre flottant et non format texte

➞ const nom_variable = contenu; // attribuer à une variable un contenu et type inmodifiable aprés sauf pour les tabeau et objets qui aurons un type de données contant mais contenu modifiable

- Afficher le contenu de la variable :

➞ document.write(nom_variable); //afficher la variable en format text sur le site

➞ alert(nom_variable); //afficher la variable dans une pop up

➞ document.write( typeof nom_variable); //afficher le type de la variable ( chaine de caractére / nombre / boolean ect... )

- Transmettre un message :

➞ document.write("le texte"); //pour afficher du texte

➞ confirm("le texte"); //ca ouvre une pop up en haut qui affiche votre texte et il faut cliquer sur ok pour continuer
